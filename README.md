# Nedit

 Dashboard for creation and mangement of buses' destination displays.

## Technologies

This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).

- TypeScript
- Context API
- React Router Dom
- Styled-components
- Theme Context
- Multilanguage
- Axios
- Formik
- Polished
- React Icons
- React Media Hook
- React toastify
- Read Excel File
- ESLint
- Prettier
- Husky
